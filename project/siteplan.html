Site Name

Name: Toyland

Reason: "Toyland" reflects a kid-friendly space where my children, Elijah and Sam, can explore creativity and fun in Minecraft. It’s simple, memorable, safe for my children and their friends. And, evokes a sense of adventure in a playful imaginative world perfect for Minecraft adventures.

Domain URI: toyland.mynetgear.com

Site Purpose

The Toyland website serves as a central hub for managing my private Minecraft servers (Bedrock and Java editions) hosted at home. It provides an easy way for my children and their friends to see a management dashboard to check server status, and pages to distribute in-game enchants, items, and gear to players using an RCON API integration.

Scenario 1

Is the server online, and can I start it if it isn't? (Eli, age 8) This will be a responsive button, either Red or Green to indicate the server status. Red for offline and the button will start the server. Green for online and will reboot the server.

Scenario 2

Can I get a diamond sword with Sharpness V for the Java server? (Frien Kobe, age 8) These will be selectors based on live queries against the server. The names of the players are pulled live. The enchantments and gear are arrays because they're fixed content values.

Typography

The typography reflects readability and a Minecraft-inspired style:

  • Font: "Roboto" - Used for body text for clean readability.
  • Font: "Mojangles" - Used for headings to mimic Minecraft’s pixelated text style.

Color Schema

The color scheme is inspired by Minecraft’s aesthetic—earthy tones with a modern twist:

  • #2F4858 (Dark Slate): Used for headers and navigation backgrounds.
  • #F5E050 (Golden Yellow): Used for accents, buttons, and highlights.
